# 法学引用类型详细说明

## 一、法律文件引用

### 1.1 法律（Law）
**识别特征：**
- 《法律名称》第X条
- 根据XXX法第X条
- XXX法第X条规定

**格式要求：**
- **首次引用：** 全称 + 发文字号/通过信息 + 简称声明
- **后续引用：** 使用简称

**示例：**
```
首次引用：
正文：《中华人民共和国民法典》（以下简称《民法典》）
脚注：《中华人民共和国民法典》，2020年5月28日第十三届全国人民代表大会第三次会议通过。

后续引用：
正文：根据《民法典》第657条...
脚注：《民法典》第657条。
```

### 1.2 行政法规（Administrative Regulation）
**识别特征：**
- XXX条例
- XXX办法
- XXX规定
- 国务院令

**格式要求：**
```
《行政法规名称》，发文字号，发布日期。

示例：
《住房公积金管理条例》，国务院令第350号，2002年3月24日修订。
```

### 1.3 司法解释（Judicial Interpretation）
**识别特征：**
- 最高人民法院关于XXX的解释
- 法释〔年份〕XX号

**格式要求：**
```
《司法解释名称》，发文字号。

示例：
《最高人民法院关于适用〈中华人民共和国民法典〉总则编若干问题的解释》，法释〔2020〕22号。
```

## 二、学术文献引用

### 2.1 著作（Monograph）
**识别特征：**
- XXX著
- XXX教授认为
- 参见XXX

**格式要求：**
```
作者：《书名》，出版社版次，页码。

示例：
王利明：《民法学》，中国人民大学出版社2020年第8版，第234页。
```

### 2.2 期刊论文（Journal Article）
**识别特征：**
- 载《期刊名》
- 发表于XXX

**格式要求：**
```
作者：“文章名”，载《期刊名》年份卷期，页码。

示例：
张三：“论合同的效力”，载《法学研究》2020年第3期，第45页。
```

### 2.3 学位论文（Thesis/Dissertation）
**识别特征：**
- 博士学位论文
- 硕士学位论文

**格式要求：**
```
作者：“论文题目”，学位授予单位年份，学位论文类型，页码。

示例：
李四：“民法典时代的合同解除制度研究”，中国政法大学2020年博士学位论文，第56页。
```

## 三、案例引用

### 3.1 指导性案例（Guiding Case）
**识别特征：**
- 指导案例第X号

**格式要求：**
```
指导案例第X号：案例名称。

示例：
指导案例第1号：上海中原物业顾问有限公司诉陶德华居间合同纠纷案。
```

### 3.2 判决书（Judgment）
**识别特征：**
- XXX判决书
- （年份）XXX号

**格式要求：**
```
当事人诉当事人，案号，法院名称，判决日期。

示例：
示例原告诉示例被告合同纠纷案，示例法院（20XX）XX民初XXXXX号，20XX年XX月XX日判决。
```

## 四、国际条约引用

### 4.1 国际公约（Convention）
**识别特征：**
- XXX公约
- United Nations Convention on XXX

**格式要求：**
```
《公约名称》，通过日期，生效日期。

示例：
《联合国国际货物销售合同公约》，1980年4月11日通过，1988年1月1日生效。
```

### 4.2 双边条约（Bilateral Treaty）
**识别特征：**
- XXX协定
- XXX议定书

**格式要求：**
```
《条约名称》，签订日期。

示例：
《中华人民共和国和美利坚合众国关于民事司法协助的协定》，1994年12月19日签订。
```

## 五、官方文件引用

### 5.1 立法文件（Legislative Document）
**识别特征：**
- 立法说明
- 草案说明

**格式要求：**
```
《文件名称》，发布机关，发布日期。

示例：
《关于〈中华人民共和国民法典（草案）〉的说明》，全国人大常委会，2020年5月22日。
```

### 5.2 政府文件（Government Document）
**识别特征：**
- 国发〔年份〕XX号
- 政发〔年份〕XX号

**格式要求：**
```
《文件名称》，发文字号。

示例：
《国务院关于加强法治政府建设的意见》，国发〔2010〕33号。
```

## 六、电子文献引用

### 6.1 网络资源（Internet Source）
**识别特征：**
- http://
- www.
- 访问日期

**格式要求：**
```
作者：“文章名”，网站名，网址，访问日期。

示例：
王五：“民法典的时代意义”，中国法学网，http://www.iolaw.org.cn，访问日期：2020年6月1日。
```

## 七、引用基本原则

### 7.1 首次引用与后续引用
**首次引用：**
- 使用全称
- 标注发文字号/通过信息
- 括注简称（如需）

**后续引用：**
- 使用简称
- 直接标注条文
- **重要：** 中文文献不使用"前引"、"同上"

### 7.2 重复引用同一文献
**中文文献：**
- ✅ 重复写完整脚注内容
- ❌ 不使用"前引（1）"或"同上"

**英文文献：**
- ✅ 可使用`supra`或`Ibid.`
- ❌ 中文文献不适用此规则

### 7.3 直接引用与间接引用
**直接引用：**
- 正文加引号
- 脚注不用引领词
- 示例：根据《民法典》第657条，"赠与合同是..."^1

**间接引用：**
- 正文不加引号
- 脚注加"参见"
- 示例：有学者认为赠与合同具有无偿性^2（参见XXX）

## 八、常见问题

### Q1: 如何处理多层级法律文件？
```
答：从大到小，逐级标注。

示例：
《民法典》第一编总则，第一章基本规定，第1条。
```

### Q2: 引用条文中的款、项如何标注？
```
答：使用阿拉伯数字，不用汉字数字。

正确：第658条第2款
错误：第658条第二款

正确：第657条第1项
错误：第657条第一项
```

### Q3: 同时引用多条如何处理？
```
答：使用顿号或"及"连接。

示例：
《民法典》第657、658条
《民法典》第657条及第658条
```

### Q4: 引用修订前后的法律如何区分？
```
答：注明修订时间或版本。

示例：
《合同法》（1999年）
《合同法》（已废止）
《民法典》（现行有效）
```

## 九、引用检索模式

### 9.1 法律条文检索模式
```python
patterns = [
    r'《([^》]+)》第(\d+)条',
    r'根据([^\s，。]+)第(\d+)条',
    r'([^\s，。]+)第(\d+)条规定',
    r'依照([^\s，。]+)第(\d+)条',
]
```

### 9.2 司法解释检索模式
```python
patterns = [
    r'《([^》]+)》.*解释',
    r'最高人民法院关于(.+?)的解释',
    r'法释〔(\d+)〕(\d+)号',
]
```

### 9.3 学术文献检索模式
```python
patterns = [
    r'([^\s，。]+)教授认为',
    r'([^\s，。]+)先生指出',
    r'([^\s，。]+)学者主张',
    r'参见([^\s，。]+)',
    r'详见([^\s，。]+)',
]
```

### 9.4 案例检索模式
```python
patterns = [
    r'(\d{4})年(\d+)月(\d+)日.*判决',
    r'(〔\d{4}〕\d+)号判决',
    r'指导案例第(\d+)号',
]
```
