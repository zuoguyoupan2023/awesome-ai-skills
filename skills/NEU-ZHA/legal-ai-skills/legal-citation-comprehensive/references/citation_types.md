# 法学引注类型详细说明

本文档详细说明各类法学文献的引注格式，基于《法学引注手册（第二版）》。

---

## 一、中文纸质出版文献

### 1.1 中文专著（第25-39条）

#### 基本格式
```
作者：《书名》，出版社年份年版，第X页。
```

#### 完整要素
1. **作者**：姓名全称，多个作者用顿号隔开
2. **著作方式**：著/主编/编（主编、编在书名后）
3. **书名**：用书名号《》，副标题用冒号连接
4. **版次**：第1版不标，第2版及以上标注（第X版）
5. **出版社**：出版社名称
6. **年份**：出版年份
7. **页码**：引用具体页码时标注

#### 示例
```
王名扬：《美国行政法》，中国法制出版社1995年版，第18页。

张新宝：《侵权责任法》（第5版），中国人民大学出版社2020年版，第73-75页。

高鸿钧、程汉大主编：《英美法原论》，北京大学出版社2013年版，第二章"英美判例法"。
```

#### 常见变体

**多人合著：**
```
作者一、作者二：《书名》，出版社年份年版。
```

**主编作品：**
```
主编主编：《书名》，出版社年份年版。
```

**章节引用：**
```
作者：《书名》，出版社年份年版，第X章"章名"。
```

---

### 1.2 中文译著（第29条、第33条）

#### 基本格式
```
[国别]原作者：《书名》，译者译，出版社年份年版，第X页。
```

#### 完整要素
1. **国别**：用方括号标注[美]/[德]/[日]等
2. **原作者**：外文姓名中文译名
3. **书名**：中文译名用书名号
4. **译者**：译者姓名+"译"
5. **出版社、年份、页码**：同中文专著

#### 示例
```
[美]富勒：《法律的道德性》，郑戈译，商务印书馆2005年版。

[德]哈特穆特·毛雷尔：《行政法学总论》，高家伟译，法律出版社2000年版，第25页。
```

#### 注意事项
- 国别标注在作者姓名之前
- 译者信息在书名之后
- 有多个译者时用顿号隔开

---

### 1.3 期刊论文（第40-42条）

#### 基本格式
```
作者：《论文题目》，载《期刊名》年份年第X期，第X页。
```

#### 完整要素
1. **作者**：姓名全称
2. **论文题目**：用书名号《》
3. **期刊名**：用书名号《》，前加"载"
4. **年份**：发表年份
5. **期数**：第X期
6. **页码**：论文在期刊中的页码

#### 示例
```
季卫东：《法律程序的意义——对中国法制建设的另一种思考》，载《中国社会科学》1993年第1期，第91页。

王保树：《股份有限公司机关构造中的董事和董事会》，载《法学研究》1994年第2期，第45-52页。
```

#### 注意事项
- 必须用"载"字连接
- 年份和期数之间用"年第"连接
- 页码是论文所在页码，非引用页码

---

### 1.4 文集论文（第9条、第47条）

#### 基本格式
```
作者：《文章名》，载编者主编：《书名》，出版社年份年版，第X页。
```

#### 完整要素
1. **作者**：论文作者
2. **文章名**：论文题目
3. **编者**：文集编者+主编/编
4. **书名**：文集名称
5. **出版社、年份、页码**：同专著

#### 示例
```
王保树：《股份有限公司机关构造中的董事和董事会》，载梁慧星主编：《民商法论丛》第1卷，法律出版社1994年版，第110页。

[德]莱纳·沃尔夫：《风险法的风险》，陈霄译，载刘刚编译：《风险规制：德国的理论与实践》，法律出版社2012年版。
```

#### 注意事项
- 先写论文作者和文章名
- 用"载"连接编者信息
- 编者信息后加主编/编

---

### 1.5 报纸文章（第43条）

#### 基本格式
```
作者：《文章名》，载《报纸名》年月日，第X版。
```

#### 完整要素
1. **作者**：姓名
2. **文章名**：题目
3. **报纸名**：报纸全称
4. **日期**：年月日
5. **版次**：第X版

#### 示例
```
何海波：《判决书上网》，载《法制日报》2000年5月21日，第2版。
```

---

### 1.6 古籍文献（第31条、第34条、第44-46条）

#### 经典古籍（无需版本信息）
```
《书名·篇名》。
```

**示例：**
```
《论语·述而》。
```

#### 整理本古籍
```
（朝代）作者：《书名》，卷次《篇名》，整理者点校，出版社年份年版，第X页。
```

**示例：**
```
（宋）王溥：《唐会要》，卷三十九《定格令》，上海古籍出版社2006年版，第825页。

（清）沈家本：《历代刑法考》，邓经元、骈宇骞点校，中华书局1985年版。
```

#### 注意事项
- 朝代用圆括号标注
- 整理者信息在书名之后
- 点校/整理/校注等需标明

---

## 二、网络文献（第50-56条）

### 2.1 网页/网站

#### 基本格式
```
作者：《文章名》，载网站名年月日，网址，访问日期访问。
```

或

```
网站名称，网址，访问日期访问。
```

#### 示例
```
梁秋坪、郝萍：《全国打击治理农村赌博工作现场会召开》，载人民网2024年10月12日，http://society.people.com.cn/n1/2024/1012/c1008-40337761.html。

法国最高行政法院网站，https://conseil-etat.fr/zh，2024年10月8日访问。
```

### 2.2 微信公众号文章

#### 基本格式
```
作者：《文章名》，载微信公众号"公众号名"年月日。
```

#### 示例
```
刘松山：《失信惩戒立法的三大问题》，载微信公众号"中国法律评论"2019年11月19日。
```

### 2.3 网络文献注意事项

1. **优先引用原始文献**：网络文章如有纸质版本，优先引用纸质版本
2. **确保来源可靠**：谨慎引用社交媒体内容
3. **保留访问记录**：注明访问日期
4. **链接处理**：链接过长时可省略，但需确保可追溯

## 三、学位论文与会议论文

### 3.1 学位论文（第64条）

#### 基本格式
```
作者：《论文题目》，学校年份年学位论文，第X页。
```

#### 完整要素
1. **作者**：姓名
2. **论文题目**：书名号的论文名
3. **学校**：授予学位的学校
4. **年份**：毕业年份
5. **学位类型**：博士学位论文/硕士学位论文
6. **页码**：引用页码

#### 示例
```
李松锋：《游走在上帝与凯撒之间——美国宪法第一修正案中的政教关系研究》，中国政法大学2013年博士学位论文，第30页。
```

### 3.2 会议论文（第63条）

#### 基本格式
```
作者：《论文题目》，会议名称会议论文，年月日于地点。
```

#### 完整要素
1. **作者**：姓名
2. **论文题目**：题目
3. **会议名称**：会议全称
4. **会议论文**：标注"会议论文"
5. **时间**：年月日
6. **地点**：会议地点

#### 示例
```
习超：《证券监管有偏私吗？》，清华大学法学院、社会科学学院、数据科学研究院"迈向数据法学"研讨会会议论文，2017年12月23日于北京。
```

## 四、法律文件（第66-80条）

### 4.1 法律条文（第66-71条）

#### 基本格式
```
《法律名称》第X条第X款第X项。
```

#### 完整要素
1. **法律名称**：用书名号《》
2. **条号**：阿拉伯数字
3. **款、项、目**：如引用，用阿拉伯数字
4. **修改年份**（如适用）：(年份年修订/修正)
5. **废止标注**（如适用）：(已废止)

#### 示例
```
《民法典》第1224条第1款第2项。

《行政处罚法》第32条。

《公司法》（2005年修订）第16条。

《公司法》（2013年修正）第36条。

《刑法》第17条之一。
```

#### 民法作业脚注补全文规则

如果正文只是提到法条编号或只概括法条含义，而没有完整列明条文原文，脚注应补足该条原文，尤其是作业答案需要读者直接看到裁判依据时。

**示例：**
```
《中华人民共和国民法典》第153条：“违反法律、行政法规的强制性规定的民事法律行为无效。但是，该强制性规定不导致该民事法律行为无效的除外。违背公序良俗的民事法律行为无效。”
```

如果正文已经完整引用条文原文，且脚注没有进一步信息，通常不必重复设脚注。

后续再次引用同一法条时，可直接写简称条号（如`《民法典》第153条第1款。`）；在课程作业中，如首次脚注已完整列明条文全文，也可写`同前注1。`或`同前注〔1〕。`

#### 直接引用与参见

- 直接引用法条、书籍、判决原文：正文使用弯引号，脚注不加"参见"。
- 概括、转述、借鉴观点或裁判思路：脚注使用"参见"。
- 转引他人已引用的资料：使用"转引自"。
- 引注支持整句时，标号放在句末标点之后；只支持句中某个词、法条或文献时，标号紧随被注释对象之后。

#### 注意事项

**数字用法（第70条）：**
- 条、款、项、目用阿拉伯数字
- 省略括号：第1款（非"第（一）款"）
- "之一""之二"保留中文

**修改版本（第69条）：**
- 引用修改过的法律需标注修改年份
- 格式：《法律》(年份年修订/修正)

**已失效法律（第69条）：**
- 标注"已废止"
- 格式：《法律》(文号，已废止)

### 4.2 规范性文件（第72-75条）

#### 基本格式
```
《文件名称》（文号），章节/条款。
```

或

```
制定机关《文件名称》（文号），章节/条款。
```

#### 完整要素
1. **制定机关**：发文机关（必要时）
2. **文件名称**：用书名号
3. **文号**：（机关简称〔年份〕序号）
4. **章节/条款**：引用的具体位置

#### 示例
```
《国务院关于在全国建立农村最低生活保障制度的通知》（国发〔2007〕19号），第四节。

《最高人民法院关于适用〈中华人民共和国行政诉讼法〉的解释》（法释〔2018〕1号），第100条。
```

#### 文号格式
- 格式：（机关简称〔年份〕序号）
- 示例：（国发〔2007〕19号）、（法释〔2018〕1号）

### 4.3 党内法规（第76条）

#### 基本格式
```
《法规名称》，年月日会议通过。
```

#### 示例
```
《中共中央关于全面推进依法治国若干重大问题的决定》，2014年10月23日中国共产党第十八届中央委员会第四次全体会议通过。
```

### 4.4 政府工作报告（第77条）

#### 基本格式
```
领导人：《报告名称》，年月日在会议。
```

#### 示例
```
李克强：《政府工作报告》，2023年3月5日在第十四届全国人民代表大会第一次会议上。
```

### 4.5 立法说明（第79条）

#### 基本格式
```
说明人：《说明标题》，载《来源》年份年第X期，第X页。
```

#### 示例
```
信春鹰：《关于〈中华人民共和国行政诉讼法修正案（草案）〉的说明》，载《全国人民代表大会常务委员会公报》2014年第6期，第887页。
```

## 五、司法案例（第87-91条）

### 5.1 一般案例

#### 基本格式
```
当事人诉当事人案，法院（年份）字号判决书。
```

#### 完整要素
1. **当事人**：原告诉被告
2. **法院**：审理法院
3. **年份**：判决年份
4. **字号**：案号
5. **判决书类型**：民事/刑事/行政判决书

#### 示例
```
示例原告诉示例被告合同纠纷案，示例法院（20XX）XX民初XXXXX号民事判决书。
```

### 5.2 公报案例（第88条）

#### 基本格式
```
当事人诉当事人案，载《最高人民法院公报》年份年第X期。
```

#### 示例
```
陆红霞诉南通市发展和改革委员会政府信息公开答复案，载《最高人民法院公报》2015年第11期。
```

### 5.3 指导案例（第89条）

#### 基本格式
```
指导案例第X号。
```

**首次引用：**
```
指导案例第X号（当事人诉当事人案，法院判决书）。
```

### 5.4 人民法院案例库（第90条）

#### 基本格式
```
当事人诉当事人案，人民法院案例库编号。
```

#### 示例
```
车某玲诉朱某芳相邻关系纠纷案，人民法院案例库2024-18-2-053-001。
```

## 六、外文文献

### 6.1 英文图书（第102条）

#### 基本格式
```
Author, *Title* (Edition, Publisher Year) page.
```

或

```
Author, *Title*, Publisher, Year, p. page.
```

#### 示例
```
William P. Alford, *To Steal a Book Is an Elegant Offense: Intellectual Property Law in Chinese Civilization*, Stanford University Press, 1995, p. 98-99.
```

### 6.2 英文期刊文章（第103条）

#### 基本格式
```
Author, Title, Volume Journal page, page (Year).
```

#### 示例
```
Charles A. Reich, The New Property, 73 Yale Law Journal 733, 737-738 (1964).
```

### 6.3 英文报纸文章（第104条）

#### 基本格式
```
Author, Title, Newspaper, Date, at page.
```

#### 示例
```
Louis D. Brandeis, What Publicity Can Do, Harper's Weekly, 20 December 1913, p. 10.
```

### 6.4 英文案例（第105条）

#### 基本格式
```
Case Name, Volume Reporter page (Court Year).
```

#### 示例
```
Chevron U.S.A., Inc. v. Natural Resources Defense Council, Inc., 467 U.S. 837 (1984).
```

### 6.5 英文再次引用（第106条）

#### 格式
```
Author, *supra* note 脚注号, at 页码.
```

或

```
*Ibid*.
```

#### 示例
```
Reich, *supra* note 24, at 738.

*Ibid*.
```

### 6.6 法文文献（第108条）

#### 图书格式
```
Auteur, *Titre*, édition, Éditeur, Année, p. page.
```

#### 示例
```
Jacques Chevallier, *L'Etat de droit*, 4° éd., Montchrestien, 2003, p. 16-29.
```

### 6.7 德文文献（第109条）

#### 图书格式
```
Autor, *Titel*, Band, Auflage, Verlag, Jahr, § Abs. Rn. Nummer.
```

#### 示例
```
Claus Roxin, *Strafrecht Allgemeiner Teil*, Band I, 4. Aufl., C.H. Beck, 2006, § 15 Rn. 19.
```

### 6.8 日文文献（第112条）

#### 图书格式
```
著者『書名』（出版社,出版年）頁数頁。
```

#### 示例
```
我妻栄『新訂担保物権法(民法講義Ⅲ)』（有斐閣,1971年）50頁。
```

## 七、再次引用格式

### 7.1 中文文献再次引用（第13条）

#### 连续引用（同页或相邻页）
```
同前注〔X〕，第Y页。
```

#### 非连续引用
```
作者书，第Y页。
```

或

```
作者文，第Y页。
```

#### 示例
```
同前注〔16〕，第3页。

王名扬书，第25页。

张文显、于宁文，第69页。
```

### 7.2 英文文献再次引用（第106条）

#### 格式
```
Author, *supra* note 脚注号, at 页码.
```

#### 同一文献连续引用
```
*Ibid*.
```

或

```
*Ibid*., at 页码.
```

## 八、引注要素优先级

### 8.1 必需要素（不可省略）

| 文献类型 | 必需要素 |
|---------|---------|
| 中文专著 | 作者、书名、出版社、年份 |
| 中文译著 | 原作者、书名、译者、出版社、年份 |
| 期刊论文 | 作者、论文名、期刊名、年份期数 |
| 法律条文 | 法律名称、条号 |
| 司法案例 | 当事人、法院、案号 |
| 网络文献 | 作者/网站、文章名、网址、日期 |

### 8.2 重要要素（尽量提供）

| 文献类型 | 重要要素 |
|---------|---------|
| 中文专著 | 版次、页码 |
| 期刊论文 | 页码 |
| 法律条文 | 修改年份、款项目 |
| 司法案例 | 判决书类型 |

### 8.3 可选要素（视情况）

- 副标题
- 丛书信息
- 系列编号
- 注释说明
