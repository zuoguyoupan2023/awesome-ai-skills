# 法学引注常见错误对照表

本文档汇总法学引注中的常见错误，基于《法学引注手册（第二版）》。

---

## 一、格式错误

### 1.1 数字用法错误

| 错误 | 正确 | 规则依据 |
|------|------|---------|
| 《民法典》第六百五十七条 | 《民法典》第657条 | 第70条：使用阿拉伯数字 |
| 第（一）条 | 第1条 | 第70条：省略括号 |
| 第一百二十条 | 第120条 | 第70条：使用阿拉伯数字 |
| 第17条第（一）款 | 第17条第1款 | 第70条：款用阿拉伯数字 |
| 第1款第（一）项 | 第1款第1项 | 第70条：项用阿拉伯数字 |
| 第25—30页 | 第25-30页 | 使用一字线连接 |
| 二〇二〇年 | 2020年 | 年份用阿拉伯数字 |

**例外情况（保留中文）：**
- 法律增订条文：第17条之一、第17条之二
- 法律文件名称中的条款序数：《刑法第六十四条》

### 1.2 标点符号错误

| 位置 | 错误 | 正确 | 规则依据 |
|------|------|------|---------|
| 作者与书名 | 王名扬《美国行政法》 | 王名扬：《美国行政法》 | 第25条：用冒号 |
| 书名与出版信息 | 《美国行政法》中国法制出版社 | 《美国行政法》，中国法制出版社 | 第25条：用逗号 |
| 出版社与年份 | 中国法制出版社，1995年 | 中国法制出版社1995年 | 第38条：不加逗号 |
| 年份与"版" | 1995年版 | 1995年版 | 正确 |
| 多个作者 | 王名扬，张新宝 | 王名扬、张新宝 | 第25条：用顿号 |
| 页码前 | 第18页 | ，第18页 | 第39条：用逗号 |
| 连续页码 | 第18页-25页 | 第18-25页 | 第39条：简洁写法 |

| 错误 | 正确 | 说明 |

## 三、再次引用错误

### 3.1 中文文献再次引用

| 错误 | 正确 | 说明 |
|------|------|------|
| 前引(1)，第5页 | 同前注〔3〕，第5页 | 中文不用"前引" |
| 同上注〔3〕 | 同前注〔3〕 | 规范用语是"同前注" |
| Ibid., at 5. | 同前注〔3〕，第5页 | 中文不用Ibid. |
| Supra note 3, at 5. | 同前注〔3〕，第5页 | 中文不用supra |
| 王名扬：前揭书，第25页 | 王名扬书，第25页 | 不用"前揭"，用简化形式 |
| 同一法条再次写"前引法条" | 同前注1。/同前注〔1〕。/《民法典》第153条第1款。 | 民法作业中再次引用法条可用同前注或重复简称 |

### 3.2 英文文献再次引用

| 错误 | 正确 | 说明 |
|------|------|------|
| 同前注 | Reich, *supra* note 24, at 738. | 英文用supra |
| Reich, note 24, at 738. | Reich, *supra* note 24, at 738. | 缺少*supra* |
| Reich, *supra*, at 738. | Reich, *supra* note 24, at 738. | 缺少note编号 |

## 六、引注位置错误

### 6.1 引注符号位置

| 错误 | 正确 | 说明 |
|------|------|------|
| 根据《民法典》^1规定，合同有效。 | 根据《民法典》规定，合同有效。^1 | 注释整句观点时，引注符号在句末标点后 |
| 《民法典》^1第657条规定 | 《民法典》第657条^1规定 | 只注释具体法条时，标号紧随被注释对象 |
| ——^1 | ^1—— | 引注符号在破折号前 |

### 6.2 引领词使用

| 情形 | 错误 | 正确 | 说明 |
|------|------|------|------|
| 直接引用 | 参见王名扬：《美国行政法》... | 见王名扬：《美国行政法》... | 直接引用用"见" |
| 概括引用 | 见王名扬：《美国行政法》... | 参见王名扬：《美国行政法》... | 概括引用用"参见" |
| 转引 | 见[美]富勒：《法律的道德性》... | 转引自[美]富勒：《法律的道德性》... | 转引用"转引自" |

### 6.3 民法作业脚注内容错误

| 错误 | 正确 | 说明 |
|------|------|------|
| 正文未引条文原文，脚注只写`《民法典》第153条。` | 脚注写明第153条全文 | 助教批注明确要求：正文未说明法条内容时，脚注应写明法条内容 |
| 直接引用法条原文却写`参见《民法典》第153条` | `《民法典》第153条：“……”` | 直接引用不用"参见" |
| 概括朱庆育观点但不写`参见` | `参见朱庆育：《民法总论》（第2版），北京大学出版社2016年版，第二十五节“强制秩序与法律行为”。` | 概括观点用"参见" |
| 脚注中的嵌套引号写成直引号 | `“违反‘打黑工’禁令的合同”` | 中文法学作业使用弯引号，外双内单 |

## 七、检查清单

### 提交前必查项

- [ ] **数字用法**：条号、页码使用阿拉伯数字
- [ ] **标点符号**：冒号、逗号、顿号使用正确
- [ ] **书名号**：所有书名、期刊名、法律名用书名号
- [ ] **必需要素**：作者、书名/文章名、出版社/期刊、年份完整
- [ ] **版次**：再版书标注版次
- [ ] **页码**：引用具体内容时标注页码
- [ ] **再次引用**：使用"同前注"或简化形式，不用"前引""同上"
- [ ] **法条全文**：正文只提法条编号时，脚注是否补足条文原文
- [ ] **直接/间接引用**：直接引用不用"参见"，概括转述用"参见"
- [ ] **外文文献**：书名斜体，格式符合该语种规范
- [ ] **引注位置**：符号在标点之后（破折号、省略号除外）
- [ ] **引领词**：根据引用类型正确使用"见"/"参见"/"转引自"

### 常见遗漏检查

- [ ] 译著是否标注国别和译者
- [ ] 论文是否加"载"字
- [ ] 文集论文是否注明编者
- [ ] 法律是否标注修改年份（修改过的法律）
- [ ] 案例是否注明法院和案号
- [ ] 网络文献是否注明网址和访问日期
