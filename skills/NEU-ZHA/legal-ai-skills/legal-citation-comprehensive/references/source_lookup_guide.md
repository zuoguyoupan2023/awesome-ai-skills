# Source Lookup Guide

Use this file whenever a citation is incomplete. Tell the user exactly where to find missing facts. Do not guess.

## Books

- Author: cover, title page, copyright page.
- Title: cover or title page; use the official title, including subtitle if needed.
- Publisher: copyright page.
- Year: copyright page. Use publication year; do not infer from file modified date.
- Edition: copyright page. Do not mark first edition unless the handbook/source specifically requires it.
- Page: printed page number. If using a searchable PDF, distinguish PDF page from printed page.

## Journal Articles

- Author and title: first page of the article.
- Journal name: first page header/footer, cover page, or database metadata.
- Year, issue, volume: first page, journal cover, or database metadata.
- Pinpoint page: printed page cited, not merely PDF page number.

## Edited Collections

- Article author/title: article first page.
- Editor and book title: book title page or table of contents.
- Publisher/year/edition: copyright page.
- Volume: cover, spine, title page, or table of contents.

## Statutes and Legal Documents

- Full name, article number, and article text: official legal database, PKULaw, NPC database, or the supplied statute file.
- Revision year and validity: legal database metadata.
- Document number: title page, issuing notice, or legal database metadata.
- For Civil Law homework: if正文 only mentions an article number, retrieve and include the article text in the footnote.

## Judicial Cases

- Case name: document title or database title.
- Court: heading area of the judgment/ruling.
- Docket number: heading area, usually in parentheses.
- Document type: judgment, ruling, decision, reply, guiding case, bulletin case, or People’s Court Case Library entry.
- Source: public bulletin, guiding case number, case library number, database URL, or original court document.

## Web Sources

- Author: byline, account name, institution name.
- Title: page title.
- Site/platform: website name or platform account.
- Publication date: page metadata or visible publication date.
- URL: canonical URL where possible.
- Access date: today’s date unless the user specifies another actual access date.

## When Source Is Not Available

Return a placeholder citation and checklist:

```text
建议格式：
作者：《书名》，[待补: 出版社][待补: 年份]年版，第18页。

需补信息：
- 出版社：请查版权页
- 年份：请查版权页
```
