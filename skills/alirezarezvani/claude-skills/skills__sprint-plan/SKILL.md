# /sprint-plan

Create a sprint plan with prioritized stories and capacity guardrails.

## Usage

```bash
/sprint-plan <goal> [capacity]
```

## Output Structure

- Sprint goal
- Committed scope
- Stretch scope
- Risks and dependencies
- Story-level acceptance criteria checks
