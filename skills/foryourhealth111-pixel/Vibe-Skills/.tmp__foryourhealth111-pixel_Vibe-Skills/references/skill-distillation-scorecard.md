# Skill Distillation Scorecard

The scorecard determines whether a source fragment is worthy of governed harvesting.

| Dimension | Healthy signal |
|---|---|
| `signal_density` | the source fragment adds non-duplicated value |
| `canonical_fit` | there is a clear canonical destination inside VCO |
| `overlap_risk` | redundancy against existing skills is explicitly low or documented |
| `no_go_boundary` | the fragment states what must not be absorbed |
| `maintenance_cost` | the harvested artifact remains small enough to govern |

Only high-signal, bounded fragments should survive harvest review.
