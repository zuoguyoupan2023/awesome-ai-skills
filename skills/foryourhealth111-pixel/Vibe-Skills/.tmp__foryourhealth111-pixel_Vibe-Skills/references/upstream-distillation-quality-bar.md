# Upstream Distillation Quality Bar

一条上游价值切片只有同时满足以下条件，才能从 `candidate` 升级到 `absorbed`：

1. 能清楚指出 target plane / canonical owner
2. 不会引入第二 orchestrator / 第二默认执行 owner / 第二 canonical truth-source
3. 已落地到至少一种正式资产（docs / config / reference / script / gate）
4. 已声明 explicit non-goals
5. 已进入 verify 或 release evidence 链
