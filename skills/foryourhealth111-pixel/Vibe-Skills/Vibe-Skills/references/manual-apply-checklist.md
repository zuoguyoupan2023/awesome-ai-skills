# Manual Apply Checklist

An apply attempt is valid only when the operator can answer every checklist item positively.

## Checklist

- The change is explicitly initiated with an apply/write flag.
- The target file and rollback path are named before the write occurs.
- Promotion-board and release-train implications are reviewed first.
- The proposal stays within bounded adjustments and no-go rules.
- The operator records evidence refs together with the apply decision.
