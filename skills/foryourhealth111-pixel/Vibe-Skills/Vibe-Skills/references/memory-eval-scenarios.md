# Memory Evaluation Scenarios

The scenarios below define the minimum quality-eval corpus for memory-runtime-v3 extensions.

| Scenario | Focus | Expected healthy outcome |
|---|---|---|
| `preference_write_precision` | mem0 preference admission | write remains confirm-gated and replayable |
| `owner_boundary_preservation` | canonical truth-source boundary | `state_store / Serena / ruflo / Cognee` remain authoritative |
| `letta_contract_mapping` | Letta vocabulary mapping | contract blocks stay advisory and non-authoritative |
| `shadow_fallback` | rollback-first fallback | ambiguous writes revert to shadow without data loss |
| `compaction_integrity` | summary / folding quality | compaction never overwrites project decisions |

Each scenario must emit evidence refs, reviewer identity, and last-reviewed date.
