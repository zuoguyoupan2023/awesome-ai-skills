# Video Guide

## Styles

### Info (Informational)

- Factual, structured presentation
- Clear section progression: intro → key points → summary
- Best for: product introductions, feature walkthroughs, educational content
- Visual style: clean, professional, data-oriented

### Story (Narrative)

- Storytelling approach with narrative arc
- Engaging flow: hook → build-up → climax → resolution
- Best for: brand stories, case studies, emotional topics
- Visual style: cinematic, atmospheric, character-driven

## When to Recommend Each

| User intent | Recommended style |
|-------------|------------------|
| "Introduce this product" | Info |
| "Explain how X works" | Info |
| "Tell the story of X" | Story |
| "Make a tutorial" | Info |
| "Create an engaging video about X" | Story |
| No preference stated | Default to Info |

## Video Generation Notes

- Video generation happens as a separate step after text/audio creation
- Text must be generated and complete before video can be requested
- Video generation takes longer (3-5 minutes total)
- The AI generates visuals to match the narration content
- Output is viewable at https://listenhub.ai/app/explainer

## Pacing Tips

- Shorter scripts produce faster-paced videos
- Detailed descriptions produce richer visuals
- One concept per segment works best for clarity
