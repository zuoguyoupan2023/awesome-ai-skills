# Podcast Mode Guide

## Modes

### Quick

- Short, concise overview of the topic
- Typical duration: ~5 minutes
- Best for: news summaries, brief introductions, quick takes
- Content depth: surface-level, highlights key points

### Deep

- Thorough analysis with extended discussion
- Typical duration: ~10-15 minutes
- Best for: topic deep-dives, educational content, analysis
- Content depth: explores nuances, provides examples, covers multiple angles

### Debate

- Two speakers with opposing or contrasting viewpoints
- Typical duration: ~10-15 minutes
- Requires exactly 2 speakers
- Best for: controversial topics, pros/cons analysis, opinion pieces
- Content depth: presents both sides with arguments and counterarguments

## When to Recommend Each

| User intent | Recommended mode |
|-------------|-----------------|
| "Give me a quick overview of X" | Quick |
| "I want to learn about X in depth" | Deep |
| "What are the pros and cons of X" | Debate |
| "Summarize this article" | Quick |
| "Analyze this paper/article thoroughly" | Deep |
| "Discuss both sides of X" | Debate |
| No preference stated | Default to Quick |

## Mode Constraints

- **Debate** requires exactly 2 speakers. If user selects debate with 1 speaker, prompt them to add a second speaker.
- **Quick** and **Deep** work with 1 or 2 speakers.
