# Munsell — Hue, Value, Chroma

**Source:** _A Grammar of Color — Part 6: A Practical Description of the Munsell Color System with Suggestions for Its Use: Hue, Value, Chroma_
**Link:** [munsell.com/color-blog/a-grammar-of-color-definition-hue-value-chroma](https://munsell.com/color-blog/a-grammar-of-color-definition-hue-value-chroma/)

## What It Is

A direct practical explanation of the **Munsell color system** in its own terms: hue, value, and chroma as three independent dimensions for describing object colors.

## Key Points

- **Hue** is measured around a circle.
- **Value** is measured on the vertical neutral axis from black to white.
- **Chroma** is measured outward from the neutral axis as color strength.
- **Notation is structural.** A Munsell notation like `5R 6/8` names hue, value, and chroma separately.
- **The system describes object colors, not pure spectral light.** That distinction is central.

## Why This Matters for the Skill

- Munsell is one of the strongest correctives to oversimplified “artist wheel” teaching.
- It gives a practical, memorable framework for **lightness-aware color thinking**.
- Useful when the agent needs to explain why **equal hue steps do not imply equal chroma/lightness behavior**.

## Practical Use

- Use for paint, pigment, and object-color discussions.
- Use when someone needs a better conceptual framework than **HSL** or a flat hue wheel.
- Pair with Briggs and Ridgway for historically grounded modern color education.
