# Hunt & Pointer — Measuring Colour

**Source:** _Measuring Colour_
**Authors:** R. W. G. Hunt, M. R. Pointer
**Archive.org:** [archive.org/details/measuringcoloure00rwgh_0](https://archive.org/details/measuringcoloure00rwgh_0)

## What It Is

An authoritative reference on **colorimetry, measurement, illuminants, observers, and reproduction**. If Fairchild is the anchor for appearance, Hunt and Pointer are the anchor for how color gets measured and specified in the first place.

## Key Points

- **Measurement precedes appearance modeling.** Spectral data, tristimulus values, observers, illuminants, and geometry determine the raw colorimetric description.
- **Viewing conditions are not optional metadata.** Illuminant, surround, flare, and adaptation affect what measured values mean in practice.
- **Object colors and light colors are not the same problem.** The book is useful for keeping those modes of appearance conceptually separate.
- **Color reproduction is a systems problem.** Capture, characterization, conversion, and output conditions all matter.

## Why This Matters for the Skill

- Strengthens answers on **print, photography, proofing, and device characterization**.
- Good source when a user needs to understand **illuminants, observers, measurement geometry, or why D50 and D65 are different workflow assumptions**.
- Helps keep agent answers grounded when discussing **ICC workflows** or **screen-to-print** conversion.

## Practical Use

- Use when discussing **spectrophotometers**, **standard illuminants**, or **observer functions**.
- Use when a task involves **measurement accuracy**, not just visual plausibility.
- Pair with ICC references for **real production workflows**.
