# Route Guide Template

Use this structure for every route guide.

## Required order

1. `Use this when`
2. `Output should look like`
3. `Shape vocabulary`
4. `## One Big Idea`
5. `## Best Default Layout`
6. `## Must Show`
7. `## Never Draw`
8. `## Canonical Recipes`
9. `## Hard Limits`
10. `## Bad vs Better Prompt`
11. `## Cut Ruthlessly`
12. `## Minimum Viable Output`

## Allowed extras

- `## Special Rule`
- `## Fundamental Shape Mapping`
- domain-specific safety notes

Use extras only when the route genuinely needs them.
