# On-Page SEO Checklist

A condensed checklist for the 8 dimensions. Use as a pre-publish gate or a quick audit pass.

---

## Title tag

- [ ] Unique across the site
- [ ] 50 to 60 characters
- [ ] Primary query near the front
- [ ] Brand at end (only if it earns inclusion)
- [ ] Not identical to H1
- [ ] Reads as a compelling search snippet, not a list of keywords

## Meta description

- [ ] Unique across the site
- [ ] 150 to 160 characters
- [ ] Restates value, does not repeat title verbatim
- [ ] Includes a soft CTA where natural
- [ ] No keyword stuffing
- [ ] Treats it as ad copy

## Header structure

- [ ] Exactly one H1
- [ ] H1 contains or paraphrases primary query
- [ ] H2s cover the major sub-topics
- [ ] No skipped levels (H2 to H4 with no H3)
- [ ] Headers describe the section accurately
- [ ] A reader could navigate by headers alone

## Body content

- [ ] First paragraph answers the primary user intent directly
- [ ] Topical depth matches or exceeds top-ranking competitors
- [ ] Related entities and concepts appear naturally
- [ ] No keyword stuffing
- [ ] Reading level matches the audience
- [ ] Paragraphs are mobile-scannable (3 to 5 lines)
- [ ] Includes original perspective or proprietary data where possible

## Internal links

- [ ] At least 2 to 3 outbound internal links to related pages
- [ ] At least 2 to 3 inbound internal links from related pages
- [ ] Anchor text is descriptive
- [ ] Links to canonical URLs (no redirects)
- [ ] No broken internal links

## Images and media

- [ ] Every meaningful image has descriptive alt text
- [ ] Decorative images use empty alt (`alt=""`)
- [ ] File names are descriptive
- [ ] Modern format used (WebP or AVIF)
- [ ] Lazy loading on below-the-fold images
- [ ] Width and height attributes set
- [ ] Captions where they aid comprehension

## URL slug

- [ ] Lowercase, hyphen-separated
- [ ] Includes primary query naturally
- [ ] Under 60 characters where possible
- [ ] No dates unless time-bound
- [ ] No tracking parameters
- [ ] Matches site URL conventions

## On-page schema

- [ ] Appropriate Schema.org type
- [ ] Required properties filled
- [ ] Validates in Rich Results Test
- [ ] Matches what is visible on the page
- [ ] Author and publisher schema linked for content pages
- [ ] BreadcrumbList schema for nested pages

---

## Quick wins (fastest to implement, biggest typical impact)

1. Rewrite title tag for CTR
2. Rewrite meta description for CTR
3. Fix any missing or duplicate H1
4. Add 2 to 3 internal links from existing related pages
5. Add alt text to images that lack it

If a page is underperforming and you only have 30 minutes, run those five.
