# 支持的文件格式

## 可直接读取（Read 工具）

| 格式 | 扩展名 | 备注 |
|------|--------|------|
| 纯文本 | `.txt` | 直接读取 |
| Markdown | `.md` `.markdown` | 直接读取 |
| PDF | `.pdf` | Cursor 自动转文本；扫描件（纯图片）无法提取，需 OCR |
| HTML | `.html` `.htm` | 直接读取，忽略标签只提取文本内容 |
| CSV | `.csv` | 按表格内容提炼 |
| JSON | `.json` | 提炼结构化数据中的文本内容 |
| XML | `.xml` | 提炼标签内文本内容 |
| Word | `.docx` | Read 工具尝试读取；若乱码提示用户另存为 `.txt` |

## 需要用户预处理后再读取

| 格式 | 处理建议 |
|------|---------|
| `.doc`（旧版 Word） | 用 Word / WPS 另存为 `.txt` 或 `.docx` |
| `.rtf` | 用文本编辑器另存为 `.txt` |
| `.odt`（LibreOffice） | 用 LibreOffice 另存为 `.txt` |
| `.epub` | 用 Calibre 等工具转为 `.txt`，或复制章节正文粘贴 |
| `.pptx` / `.ppt` | 复制演讲稿文本粘贴，或导出为 PDF 后读取 |
| `.xlsx` / `.xls` | 复制关键数据区域粘贴，或导出为 `.csv` 后读取 |

## 网络来源（WebFetch 工具）

| 来源 | 示例 URL 格式 | 备注 |
|------|-------------|------|
| 普通网页 | `https://example.com/article` | 大多数可正常抓取 |
| 微信公众号 | `https://mp.weixin.qq.com/s/...` | 可能被拦截，拦截时需手动复制 |
| 知乎文章 | `https://zhuanlan.zhihu.com/p/...` | 通常可抓取 |
| 掘金 / 少数派等 | 各平台文章链接 | 部分平台有反爬，被拦截时手动复制 |
| 邮件 / 内网链接 | — | 无法抓取，需粘贴正文 |
